Galois Fields

In a Galois Field also known as a finite field, there is a limited specific number of elements. It simply limits the possible values. In a Galois Field there are a four operations: Addition, Subtraction, Multiplication and Division. The generalization is GF(pᴷ), where p is base and k is the exponent.  The base has to be a prime number for the system to work.

No matter how many operations you preform over a Galois Field the result will always be a number that exists inside the field.

GF(2) is the simpelest Galois Field that exists. Since the base is a prime number 2 and the exponent is 1 there only two possible values 0 and 1.  GF(3¹) has 3 possible values. and GF(11¹) has 11 posible values.

If we add an exponent the range of possible values increases vastly GF(2²) has 4 possible values, GF(2⁴) has 16 possible values.  GF(5²) has 25 possible values and GF(7⁷) has 823543 possible values.  GF(2⁸) has 256 possible values.
