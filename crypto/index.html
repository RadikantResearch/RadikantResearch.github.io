Radikant Crypto is a cryptolibrary providing cryptograhpic primivitves such as ECC, Hash and various ciphers and written in C and has no external libraries.

Introduction

Radikant Crypto is a research project and software reference implementation primarily designed for macOS. While primitives are portable to microcontrollers and amenable to hardware optimization (SIMD/NEON), this library is not intended for production use without thorough security auditing. While there are alot of libraries providing the same functionality like openSLL, WolfSLL, EmbedTLS and many more these libraries often have a high level of internal abstractions and are often tightly coupled with some type of internal core native to the library it can become difficult for the reader to focus on the code that does the work. This library is therefore kept simple and internally decoupled wherever possible so its easy just to take coponents and integrate them elsewhere.

This library is used in other appliction level code such as Radikant-TLS-C and Radikant-SQL-C
